# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started
1. Get in the react folder
   ```bash
   cd react-native-advanced
   ```
2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
    npx expo start
   ```
4. Run the app on an Android emulator
   ```bash
   Press a in the terminal
   ```
   Or run the app on a physical device
   - Download the Expo Go app from the App Store or Google Play.
   - In your terminal, press c to open the QR code.
   - Scan the QR code using the Expo Go app on your device.
